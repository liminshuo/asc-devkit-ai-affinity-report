# asc-devkit AI 亲和诊断报告

GitCode asc-devkit 算子开发文档的 AI 亲和性全量诊断报告（35 条原则实测）。

## 在线访问

- 本地：`python3 -m http.server 8899` → http://localhost:8899/
- GitCode Pages：`https://<你的用户名>.gitcode.io/asc-devkit-ai-affinity-report/`

## 部署到 GitCode Pages

1. 在 GitCode 新建公开仓库 `asc-devkit-ai-affinity-report`
2. 推送本目录到 `main` 分支
3. 仓库设置 → Pages → 启用，构建产物目录选 `public`
4. 等待 1–2 分钟，通过 Pages 地址访问

|------|------|
| `index.html` | 首页，自动跳转到报告 |
